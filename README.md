# 💥 Superpoderoses – Object Modeling in Wollok

Proyecto individual desarrollado en Wollok, enfocado en el modelado de un sistema de personajes con distintos superpoderes y comportamientos dinámicos.

## 📌 Descripción

Este proyecto simula un conjunto de personajes que poseen distintos superpoderes, cada uno con reglas y efectos propios.

El objetivo principal fue diseñar un modelo orientado a objetos donde los poderes no sean simples atributos, sino comportamientos con responsabilidad propia.

## 🧠 Conceptos aplicados

Programación Orientada a Objetos (POO)

Encapsulamiento

Delegación de comportamiento

Composición (personaje → poderes)

Uso de colecciones

Métodos como objetos de decisión

Testing automatizado

## 🏗 Decisiones de diseño

Se priorizó que:

Cada objeto tenga una responsabilidad clara.

Los poderes definan su propia lógica.

El personaje delegue comportamiento en sus superpoderes.

El sistema sea fácilmente extensible (agregar nuevos poderes sin modificar la estructura principal).

## 💻 Sobre Wollok

Wollok es un lenguaje educativo orientado a objetos utilizado para el aprendizaje de modelado y diseño de software.

Permite trabajar conceptos fundamentales como:

Objetos y clases

Mensajes

Polimorfismo

Modelado de dominio

## 🎯 Objetivo del proyecto

Fortalecer habilidades de diseño orientado a objetos, enfocándome en la correcta distribución de responsabilidades y en la construcción de modelos expresivos y extensibles.
